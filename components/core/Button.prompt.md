Primary action button for HEART — use for any clickable action; civic and restrained, never bubbly.

```jsx
<Button variant="primary" size="md" onClick={save}>Generate report</Button>
<Button variant="secondary">Export CSV</Button>
<Button variant="authority">Open exec view</Button>
```

Variants: `primary` (scarlet), `secondary` (white/border), `ghost` (transparent), `authority` (deep slate — for leadership chrome), `danger` (critical alerts). Sizes `sm | md | lg`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`.
